Här får du en beskrivning av Git och hur man skapar en repository

För att skapa en git.repository behöver du en mapp som ligger på skrivbordet
Sedan öppnar du en terminal i t.ex. VSC: terminal: New terminal.
Se till att vara i mappen som skapades när du arbetar i terminalen.

  1. När du ser adressen på din mapp i terminalen, skapar du lokalt repository genom att skriva: git init


  2. Då får du ett svar: "Initialized empty git repository".


  3. Nu kan du börja skapa innehåll i din mapp genom att välja Explorer-knappen längst upp i VSC.


  4. Högerklicka i arbetsytan som ligger till vänster och lägg till "New file" som t.ex. index.html


  5. Avsluta med kommandot: git add index.html


  6. Skapa ett repository i GitHub genom att klicka på + ikonen vid din profilbild


  7. Nu ska du tillbaka till VSC och ta fram terminalen där du skriver in ditt projektnamn t.ex. git remote add origin https://github.com/Nemetona/Technics-for-webdesign.git


  8. Gör en git push och du får git push --set-upstream origin maste

Git kommandon

Uppdatera lokal repository mot GitHub

  1. git status = kolla om det finns ändringar i lokalt projekt
  2. git checkout dittFilnamn.html = tar bort osparade ändringar i lokal fil
  3. git fetch --all = hämta senaste versionsinformationen från github
  4. git pull = ladda ner senaste ändringarna från github

Spara nya ändringar till GitHub

  1. git status = kolla om det finns ändringar i lokalt projekt
  2. git add dittFilnamn.html = lägger till ändringar från fil som ska skickas till GitHub
  3. git add . = lägger till alla ändringar från alla filer som ska skickas till GitHub
  4. git commit -m "ditt meddelande som beskriver vad du ändrat" = fungerar som en label för ändringen
  5. git push = skickar ändringarna till GitHub
