Här får du en beskrivning av Git och hur man skapar en repository

För att skapa en git.repository behöver du en mapp som ligger på skrivbordet
Sedan öppnar du en terminal i t.ex. VSC: terminal: New terminal.
Se till att vara i mappen som skapades när du arbetar i terminalen.


  1. När du ser adressen på din mapp i terminalen, skapar du lokalt repository genom att skriva: git init


  2. Då får du ett svar: "Initialized empty git repository".


  3. Nu kan du börja skapa innehåll i din mapp genom att välja Explorer-knappen längst upp i VSC.


  4. Högerklicka i arbetsytan som ligger till vänster och lägg till "New file" som t.ex. index.html

  5. Avsluta med kommandot: git add index.html
